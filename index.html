GenericKnimeNodes by roettig

Base package for GenericKnimeNodes

The GenericKnimeNodes project provides code for automatic integration of external (binary) programs into the Konstanz Information Miner (KNIME). The set of tools have to adhere to the Common Tool Description (CTD) concept that defines for each tool its input an output parameters and also allows each tool to be configured using such a CTD parameter file. The KNIME nodes are then generated automatically based on the set of CTD files and a KNIME plugin is generated.
